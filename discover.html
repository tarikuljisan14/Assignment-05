Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element, and it handles events for child elements using event bubbling. This allows for efficient event management, especially for dynamically added elements..

Question-4: What is event bubbling in the DOM?

Event Bubbling in the DOM is a process where an event starts from the target element and propagates (bubbles) up through its ancestor elements. It moves from the innermost element to the outermost (from child to parent).

Question-5: How do you create, add, and remove elements using JavaScript?

Use document.createElement() to create an element, then appendChild() or append() to add it to the DOM. To remove an element, use remove() or removeChild().
